Agenda - Dicionário de dados

Usuários

usuario_id = * id do usuario *

usuario_nome = * nome do usuario *

usuario_email = * email do usuario *

usuario_status = * status de usuario *

* Unidades: inteiro entre 0-1 *

* 0 - desativado 1 - ativado *

usuario_senha = * 10 caracteres *

Eventos

evento_id = * id do evento *

evento_data_hora = * data e horario do evento *

evento_descricao = * descricao do evento *

evento_nome = * nome do evento *

evento_status = * status de evento *

* Unidades: inteiro entre 0-1 *

* 0 - desativado 1 - ativado *

usuario_id = * id referente ao usuario que criou o evento *
