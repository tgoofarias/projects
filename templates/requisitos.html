Agenda - Requisitos

Neste documento é apresentado os requisitos do sistema de Agenda, por meio deles podemos entender das funcionalidades que o sistema necessitará para estar completo e funcional, assim capaz de cumprir seu propósito.

Requisitos Funcionais

# Requisito Descrição Prioridade Caso de Uso Relacionado
RF01 Cadastrar Usuário Usuário poderá se cadastrar no sistema Alta Manter Usuário
RF02 Login Usuário poderá se logar no sistema Alta Manter Usuário
RF03 Cadastrar Evento Usuário poderá marcar um evento Alta Manter Evento
RF04 Visualizar calendário Usuário poderá visualizar os meses Alta Visualizar calendário
RF05 Visualizar dia Usuário poderá visualizar os eventos do dia Alta Visualizar dia
RF06 Deletar evento Usuário poderá deletar eventos Média Manter eventos
RF07 Notificar evento Usuário poderá ser notificado dos eventos Baixa Notificar evento

Requisitos Não Funcionais

# Requisito Métrica
RNF01 Segurança Nenhum dado será exposto publicamente
RNF02 Desempenho Sistema não apresentará lentidão ou problemas de espaço de disco
RNF03 Facilidade de Uso Sistema deverá ser fácil manuseio e intuitivo

Regras de Negócio

# Descrição
RDN001 Usuário apenas pode acessar agenda após ser realizado o login
RDN002 Ao cadastrar um evento, é obrigatório a data e horário
RDN003 Ao realizar logout, para acessar a agenda será necessário logar novamente
